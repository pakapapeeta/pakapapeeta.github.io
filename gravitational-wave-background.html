Stochastic Gravitational Wave Background Research

Probing early universe physics through pulsar timing arrays and gravitational wave astronomy

NANOGrav & Early Universe Cosmology

Using pulsar timing array data from NANOGrav to constrain early universe models and search for primordial gravitational wave backgrounds. This groundbreaking work bridges observational astronomy with fundamental physics, offering unprecedented windows into cosmic inflation, phase transitions, and alternative cosmological scenarios.

Research Focus Areas

  • Primordial GW Detection: Searching for gravitational waves from the early universe in pulsar timing data
  • Bouncing Cosmologies: Investigating non-singular cosmological models through GW signatures
  • Phase Transitions: Constraining first-order phase transitions in the early universe
  • String Cosmology: Testing string theory predictions using gravitational wave observations
